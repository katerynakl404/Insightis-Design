Marquee

Infinite horizontal scroll for logo walls / partner strips. Decorative — disabled below 768px and under prefers-reduced-motion. Pauses on hover + when off-screen.

HTML

<div class="ins-marquee">
  <div class="ins-marquee__track">
    <!-- Content × 2 for seamless loop -->
    <img src="/logos/acme.svg" alt="Acme">
    <img src="/logos/widgets.svg" alt="Widgets">
    ...
    <!-- Duplicate set -->
    <img src="/logos/acme.svg" alt="" aria-hidden="true">
    ...
  </div>
</div>

Rules

  • Duplicate content once — the track animates from 0 to -50%
  • First set of logos have meaningful alt; duplicates use alt="" + aria-hidden="true"
  • Decorative — no animation below 768px, paused on hover, paused when off-screen (IntersectionObserver in storybook.js)
