Component

Kbd

Keyboard key visualization. Useful for command palette hints, shortcuts in docs.

Examples

⌘ ⇧ ⌥ K Enter Esc

Combos

Press ⌘ + K to open the command palette.

Use ⌘ + ⇧ + P for the query panel.

Press Esc to dismiss any modal.

HTML

<kbd>⌘</kbd> + <kbd>K</kbd>
