Accessibility

  • Icon-only buttons MUST include aria-label
  • Loading buttons set aria-busy="true" and stay focusable; add disabled on submits
  • Touch target: minimum 44×44 on mobile (enforced by .ins-btn--icon min-width/height)
  • Focus ring: --ins-focus-ring applied via :focus-visible from base.css
