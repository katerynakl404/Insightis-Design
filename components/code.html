Fenced block (default — no copy button)

sql 
SELECT
  date_trunc('month', created_at) AS month,
  SUM(amount) AS mrr
FROM subscriptions
WHERE status = 'active'
GROUP BY 1
ORDER BY 1 DESC;

Fenced block with copy button (opt-in)
