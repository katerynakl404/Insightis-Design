Component

Form group

Composition pattern for label + input + helper/error. Wires ARIA correctly so screen readers announce validation.

Variants

We'll never share your address.
Enter a valid email address.

HTML

<div class="ins-form-group">
  <label class="ins-form-group__label ins-form-group__label--required" for="email">Email</label>
  <input type="email" class="ins-input" id="email"
         required aria-describedby="email-hint">
  <span class="ins-form-group__hint" id="email-hint">We'll never share it.</span>
</div>

<!-- Error state: aria-invalid + error message via aria-describedby -->
<div class="ins-form-group">
  <label class="ins-form-group__label" for="e2">Email</label>
  <input type="email" class="ins-input is-error" id="e2"
         aria-invalid="true" aria-describedby="e2-err">
  <span class="ins-form-group__message ins-form-group__message--error"
        id="e2-err">Enter a valid email.</span>
</div>
