Layout primitives

Four low-level layout utilities: container, section, split, grid. Section-level block compositions (hero, feature grid, pricing, CTA, promo, newsletter, stats strip, team grid, sidebar+content, testimonial, logo wall, FAQ, legal prose) live in Patterns — each on its own page, built from real component classes.

Container

Max-width wrapper (1280px) with responsive horizontal padding (16 / 24 / 32).

.ins-container

Section

Vertical rhythm wrapper — padding-block scales at ≥768.

<section class="ins-section">
  <div class="ins-container">...</div>
</section>
