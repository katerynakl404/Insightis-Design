SEO habits (dev team must preserve)

✓ Always

One h1 per page. Sequential heading levels (h1→h2→h3, never skip). alt on every <img> (alt="" for decorative). Status chips pair color + icon + text.

✗ Never

Don't use <div> for clickable elements (use <button>). Don't skip heading levels. Don't convey meaning with color alone.
