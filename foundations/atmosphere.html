Atmosphere

Three optional "feel" treatments with strict scope rules. All opt-in via utility class.

1. Soft heading color

Use .ins-text-heading-soft only on display-size text placed directly on --ins-surface-page. Never on h1–h4 or inside cards.

Softened hero headline

2. Noise texture

Adds 3% opacity tactile grain via ::before. Use on marketing/brand page backgrounds only. Never on docs/legal pages.

This preview has .ins-bg-noise applied. 
<body class="ins-bg-noise">
  <!-- page content -->
</body>

3. Aurora backdrop

Multi-radial gradient for brand-focused hero sections. One per page maximum. Never in docs/legal/pricing. Uses contain: paint to isolate paint work.

Aurora hero

Use on ONE hero section per brand page.

<section class="ins-bg-aurora ins-section">
  <div class="ins-container">
    <h1 class="ins-text-display-xl">Hero</h1>
  </div>
</section>
