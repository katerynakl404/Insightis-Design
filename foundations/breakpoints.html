Breakpoints

Mobile-first. Four min-width stops. 320px is the smallest supported viewport.

NameMin widthTokenTypical device
mobile320px (floor)—iPhone SE, iPhone 5s portrait
mobile-lg480px--ins-bp-480 / --ins-breakpoint-smLarge phones portrait, small phones landscape
tablet768px--ins-bp-768 / --ins-breakpoint-mdiPad portrait, small tablets
laptop1024px--ins-bp-1024 / --ins-breakpoint-lgiPad landscape, small laptops
desktop1280px--ins-bp-1280 / --ins-breakpoint-xlLaptop, desktop (container max)

Visual map

0–479 480–767 768–1023 1024–1279 1280+

Resize your browser; layouts across the kit reflow at these stops.

Usage

/* Mobile-first: base rules target <480px */
.my-card { padding: var(--ins-space-lg); }

/* Tablet+ */
@media (min-width: 768px) {
  .my-card { padding: var(--ins-space-xl); }
}

/* Desktop */
@media (min-width: 1280px) {
  .my-card { padding: var(--ins-space-2xl); }
}
