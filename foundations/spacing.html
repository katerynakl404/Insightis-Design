Usage

✓ Do

Use semantic T-shirt aliases (--ins-space-lg) in components. Keep all values on the 4px grid.

✗ Don't

Don't use arbitrary px. Don't use 2px half-step unless a hair-line nudge is required.

padding: var(--ins-space-lg);   /* 16px */
gap: var(--ins-space-md);         /* 12px */
margin-block: var(--ins-space-3xl); /* 48px */
